# personal-website

## 中文

`personal-website` 是 Brian 的個人作品與履歷展示網站。這個專案採用靜態網站形式，核心內容由資料檔驅動，展示個人背景、研究經歷、技能與代表專案。

### 網站內容
目前網站的內容包含：
- 個人簡介
- 學經歷與研究背景
- 技能分類
- 研究論文 / 發表
- 專案展示
- 聯絡資訊

從目前資料檔來看，展示的代表專案包含：
- Knowledge Map
- AI Work System
- 3C Detox App
- Elder Life App

### 專案形式
這個 repo 目前偏向資料驅動的靜態頁面，而不是 React / Vue 類型前端框架專案。

### 主要檔案
- `data.json`: 網站主要內容資料
- `style-4-notion-warm.html`: 當前的主要頁面樣式版本
- `images/`: 圖片資產
- `projects/`: 各專案的落地頁或入口頁

### 技術棧
- HTML
- CSS
- JSON 資料驅動內容
- GitHub Pages 友善的靜態部署模式

### Current site content
The current site includes:

- academic / research background
- categorized skills
- research papers / publications
- project showcase

Based on the current data file, featured projects include:

This repository currently behaves more like a data-driven static site than a framework-heavy frontend app.

### Key files
- `data.json`: main site content data
- `style-4-notion-warm.html`: current main page variant
